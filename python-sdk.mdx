## Installation

<CodeGroup>
```bash pip
pip install channel3_sdk
```

```bash uv
uv add channel3_sdk
```

</CodeGroup>

## Authentication

You can provide your API key using the `CHANNEL3_API_KEY` environment variable, or by passing it directly to the client.

```python
from channel3_sdk import Channel3, AsyncChannel3

# Initialize with environment variable
client = Channel3()

# Or, initialize with API key directly
client_with_key = Channel3(api_key="your_api_key_here")

# Async client
async_client = AsyncChannel3()
```

## Usage

All top-level endpoints are available as properties on the client. For example, to use the `search` endpoint, you can do the following:

```python
from channel3_sdk import Channel3

client = Channel3()

def main():
    response = client.search.perform(
        query="organic cotton t-shirt"
    )
    print(response)

if __name__ == "__main__":
    main()
```

### Async Usage

```python
import asyncio
from channel3_sdk import AsyncChannel3

client = AsyncChannel3()

async def main():
    response = await client.search.perform(
        query="organic cotton t-shirt"
    )
    print(response)

if __name__ == "__main__":
    asyncio.run(main())
```

## Advanced Usage

### Search with filters

```python
from channel3_sdk import Channel3

client = Channel3()

def main():
    response = client.search.perform(
        query="running shoes",
        filters=Filters(
            availability=[AvailabilityStatus.IN_STOCK],
            price=FiltersPrice(
                min_price=10,
                max_price=50,
            ),
            gender="male",
        ),
        limit=20,
    )
    print(response)

if __name__ == "__main__":
    main()
```

### Search with an image URL

```python
from channel3_sdk import Channel3

client = Channel3()

def main():
    response = client.search.perform(
        image_url="https://example.com/images/shoe.jpg",
        limit=12,
    )
    print(response)

if __name__ == "__main__":
    main()
```

### Search with a base64 image

```python
import base64
from channel3_sdk import Channel3

client = Channel3()

def main():
    with open("shoe.jpg", "rb") as f:
        b64 = base64.b64encode(f.read()).decode("utf-8")

    response = client.search.perform(
        base64_image=b64,
        limit=12,
    )
    print(response)

if __name__ == "__main__":
    main()
```

### Configure search behavior

```python
from channel3_sdk import Channel3

client = Channel3()

def main():
    response = client.search.perform(
        query="linen shirt",
        config={
            "enrich_query": False,  # default True
            "semantic_search": True,  # default True
        },
    )
    print(response)

if __name__ == "__main__":
    main()
```

### Provide request context

```python
from channel3_sdk import Channel3

client = Channel3()

def main():
    response = client.search.perform(
        query="sneakers",
        context="vip-customer-123",
        limit=10,
    )
    print(response)

if __name__ == "__main__":
    main()
```

### Enrich a product URL

```python
from channel3_sdk import Channel3

client = Channel3()

def main():
    response = client.enrich.perform(
        url="https://brand.com/products/linen-shirt"
    )
    print(response)

if __name__ == "__main__":
    main()
```

## Error Handling

When the library is unable to connect to the API, or if the API returns a non-success status code (i.e., 4xx or 5xx response), a subclass of `APIError` will be thrown.

```python
from channel3_sdk import Channel3
from channel3_sdk.errors import APIError

client = Channel3()

def main():
    try:
        client.search.perform()
    except APIError as e:
        print(f"Status code: {e.status_code}")
        print(f"Error name: {e.request.body.get('error', {}).get('type')}")
        print(f"Headers: {e.headers}")

if __name__ == "__main__":
    main()
```

Error types are exposed on the `channel3.errors` module.
