## What is the Channel3 MCP?

The Channel3 MCP gives your AI agents superpowers for product discovery. Your agent can search through tens of millions of real products in real-time, complete with current prices, rich images, detailed descriptions, and real-time availability.

Channel3's MCP server is available at `https://docs.trychannel3.com/mcp` and includes:

- **Documentation search** - Query Channel3 API documentation
- **Product search** - Search across millions of products
- **Product details** - Get comprehensive product information
- **Brand lookup** - Find brands and their information
- **Website lookup** - Find retailer information
- **Product enrichment** - Get full details from product URLs

## Authentication

When AI tools call Channel3 API endpoints that require authentication, they will prompt you to provide your API key at that moment. The API key should be provided as the `x-api-key` header.

<Note>
  Get your API key from the [Channel3 Dashboard](https://trychannel3.com/dashboard).
</Note>

Authentication is handled directly by your AI tool and not stored or processed by Mintlify.

## Available Tools

The Channel3 MCP server exposes the following tools:

| Tool | Description |
| :--- | :---------- |
| **search** | Search Channel3 documentation |
| **search_products** | Search for products with filters for brand, price, gender, availability, and categories |
| **get_product_details** | Get comprehensive details about a specific product by ID |
| **find_brand** | Find brand information by name, including commission rates |
| **find_website** | Find retailer information by URL or domain |
| **enrich_product_url** | Get full product details from a product URL |

## Example Usage

After connecting the MCP server to your AI tool:

```
You: "Find me black running shoes under $100"
AI: [Uses search_products tool to find relevant products]

You: "Tell me more about the first product"
AI: [Uses get_product_details tool to get full information]

You: "What's Nike's commission rate?"
AI: [Uses find_brand tool to look up Nike's information]
```
